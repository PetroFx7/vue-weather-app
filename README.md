# WeatherApp

Додаток погоди побудований на Vue 3 + TypeScript + Vite.

## Стек технологій

| Категорія | Технологія |
|---|---|
| UI-фреймворк | Vue 3 (Composition API, `<script setup>`) |
| Мова | TypeScript |
| Збірник | Vite |
| Стейт-менеджмент | Pinia |
| Маршрутизація | Vue Router |
| HTTP-клієнт | Axios |
| Графіки | Chart.js (без vue-chartjs, raw API) |
| Стилі | CSS змінні (без фреймворків) |

## Функціональність

### Основне

- **Пошук міста з автокомплітом** — Geocoding API OpenWeatherMap, debounce 300 мс
- **Картка поточної погоди** — температура, відчувається як, вологість, вітер, тиск, видимість, схід/захід сонця
- **Графік температури** — Chart.js, по годинах на поточний день або по днях на 5 днів
- **Перемикання "Сьогодні / 5 Днів"** — картка і графік змінюються разом
- **Декілька блоків погоди** — до 5 блоків, за замовчуванням 1, кнопка "+"
- **Видалення блоку** — модальне вікно з підтвердженням

### Вибране

- Додавання/видалення міста в обране (іконка серця на картці)
- Вкладка "Вибране" — блоки погоди без вибору міста, лише перемикання "Сьогодні / 5 Днів"
- Зберігання в `localStorage`
- Ліміт 5 міст; при перевищенні — модальне вікно зі списком поточних обраних

### Додатково

- **Перемикання "День / Ніч"** — фільтрує дані графіка по денних або нічних годинах, змінює кольорову схему чарту
- **Геолокація по IP** — автоматично визначає місто при першому завантаженні (`ipapi.co`)
- **Прелоадери** — відображаються під час кожного запиту до API
- **Багатомовність EN / UK** — перемикач у хедері, перекладаються як інтерфейс, так і описи погоди від API (параметр `lang`)
- **Темна / Світла тема** — CSS-змінні, зберігається в `localStorage`
- **Адаптивний дизайн** — контейнер 1200 px, мінімум 360 px, блоки flex-wrap

## Запуск проєкту

### 1. Встановлення залежностей

```sh
pnpm install
```

### 2. Налаштування API ключа

Скопіюйте файл прикладу та вставте свій ключ [OpenWeatherMap](https://openweathermap.org/api):

```sh
cp .env.example .env
```

`.env`:
```
VITE_OPENWEATHER_API_KEY=your_api_key_here
```

### 3. Запуск у режимі розробки

```sh
pnpm run dev
```

Додаток буде доступний за адресою `http://localhost:5173`

### 4. Перевірка типів

```sh
pnpm run type-check
```

### 5. Production-збірка

```sh
pnpm run build
```

### 6. Попередній перегляд production-збірки

```sh
pnpm run preview
```

## API

Проєкт використовує безкоштовний план [OpenWeatherMap](https://openweathermap.org/):

| Endpoint | Призначення |
|---|---|
| `/geo/1.0/direct` | Автокомпліт пошуку міста |
| `/data/2.5/weather` | Поточна погода |
| `/data/2.5/forecast` | Прогноз на 5 днів / 3 год |
| `ipapi.co/json` | Визначення міста за IP |
